# Codex 워크스페이스 안내

이 문서는 사람이 보기 쉽게 정리한 한국어 참고본입니다.
Codex의 실제 instruction discovery에는 사용되지 않습니다.
실제 적용되는 규칙은 같은 위치의 `AGENTS.md`를 기준으로 합니다.

## 범위

- 이 디렉터리에는 Codex 전용 설정, 에이전트 정의, 워크플로우가 들어 있습니다.
- 저장소 전체 규칙은 루트 `AGENTS.md`에서 관리하고, 여기에는 Codex 설정 관련 안내만 둡니다.

## 편집 원칙

- Codex 설정 변경은 작고 검토하기 쉬운 단위로 유지합니다.
- 작업상 필요하지 않다면 기존 에이전트 이름, 모델, 닉네임을 유지합니다.
- 각 에이전트의 `developer_instructions`는 역할 고유 행동에 집중시킵니다.
- 이미 상위 `AGENTS.md`에서 상속되는 공통 코딩 규칙을 각 에이전트 파일에 반복하지 않습니다.
- 새 Codex 설정을 추가할 때는 공식 `AGENTS.md` discovery 모델과 맞는 구조와 이름을 사용합니다.
